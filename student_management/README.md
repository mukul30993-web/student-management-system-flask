# Student Management System

A simple Student Management System built with Flask and SQLite.

## Features

- Add Student
- View Students
- Edit Student
- Delete Student
- Search Student
- SQLite Database
- Bootstrap UI
- Responsive Design

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap

## Installation

Clone the repository

```bash
git clone https://github.com/mukul30993-web/student-management.git
```

Go into the project folder

```bash
cd student-management
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
